About Me

Hi, my name is Jessica Meng! I am a second-year Computer Science & Business Student @ Northeastern University. My interests include UI/UX design, web development, marketing, and product management. I've created a personal project called Snoopify - Snoopy x Spotify using Swift/SwiftUI. This project was a design project that recreated Spotify's mobile app interface, but with elements of Snoopy included. I am interested in a SWE co-op during Jan - Aug 2026! Thank you!

Projects

This is a website I made for my high school international business practice firm class. The goal of the class was to create a business and "trade" inventory with businesses from other schools around the world. As the CTO of the company, I created the website and designed the menu.

Asian American Cultural Club Website

This is a website I made for my high school Asian American Cultural Club. The website was made to organize our club's events and to provide easy access to the calendar and sign-ups.

Calendar Project

This is a project I made for my Object-Oriented Design class. The project was to create a calendar application that allows users to add, edit, and delete events and series of events. The project was made using Java and Swing, featuring the model-view-controller (MVC) design pattern. It also includes headless mode, interactive mode, and a GUI mode.

Code available upon request via email at jessicammeng@gmail.com

This is a personal project I made to try out Swift/SwiftUI in Xcode. I redesigned Spotify's mobile app but created my own art and color scheme. I am obsessed with Snoopy and Olivia Rodrigo/Gracie Abrams, so I created a theme around them. The project is not affiliated with Spotify in any way.
